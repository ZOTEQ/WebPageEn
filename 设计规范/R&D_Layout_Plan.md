# R&D Laboratories 模块 — 版式重设计划

## 当前状态
- 4 个实验室卡片在 `.labs-masonry` 中按 1col → 2col → 3col 排列
- 每张卡片：图片 + 圆角容器 + 图标 + 标题 + 地点 + 描述
- 风格偏传统，缺乏视觉层次和互动记忆点

---

## 方案 A：「Split Panorama」左右分屏

**灵感**：Awwwards 级左右分屏沉浸布局

```
┌─────────────────────────────────────────┐
│   R&D Laboratories                      │
│   Four specialized research facilities  │
├──────────────────┬──────────────────────┤
│                  │  ┌─────┐  ┌─────┐   │
│   大背景图        │  │合成  │  │生物  │   │
│   (全高, 半透明   │  │香料  │  │发酵  │   │
│    渐变遮罩)      │  └─────┘  └─────┘   │
│                  │  ┌─────┐  ┌─────┐   │
│   4 个光点标记    │  │调香  │  │植物  │   │
│   对应实验室位置   │  │评香  │  │提取  │   │
│                  │  └─────┘  └─────┘   │
└──────────────────┴──────────────────────┘
```

**特点**：
- **左侧**：宽幅实验室背景图（~50% 宽度，全高），4 个可交互光点标记实验室位置
- **右侧**：2x2 紧凑卡片网格，卡片为「极简边框 + 文字」风格，无图片
- **悬停**：卡片背景色变为浅橙色，左侧对应光点放大发光
- **taste-skill 契合**：不对称布局，打破均匀网格，无泛用图片

## 方案 B：「Lab Timeline」纵向时间线

**灵感**：杂志编辑式叙事布局

```
┌──────────────────────────────────────────┐
│   R&D Laboratories                       │
│   subtitle                              │
├──────────────────────────────────────────┤
│                                          │
│  ┌──────┐  合成香料实验室                 │
│  │ 大图  │  Chongqing                    │
│  └──────┘  Advanced synthesis...         │
│  ────────●────────────────────           │
│           │                              │
│  ┌──────┐  │生物发酵实验室                 │
│  │ 大图  │ Shanghai                     │
│  └──────┘  │Cutting-edge...              │
│  ────────●────────────────────           │
│           │                              │
│  (交替左右排列，中间一条连接线)            │
│                                          │
└──────────────────────────────────────────┘
```

**特点**：
- 纵向时间线布局，4 个实验室交替排列（左→右→左→右）
- 中间一条垂直线串联，每个节点有圆点标记
- 每项：大图占 1/3，文字占 2/3，或交替方向
- **悬停**：图片放大 + 节点圆点脉动动画
- **taste-skill 契合**：叙事驱动，非对称交替，打破常规网格

## 方案 C：「Hover Reveal Grid」悬停揭示网格

**灵感**：极简主义 + 惊喜交互

```
┌──────────────────────────────────────────┐
│   R&D Laboratories                       │
│   subtitle                              │
├──────────────────────────────────────────┤
│                                          │
│  ┌──────────────────────┐               │
│  │  合成香料实验室        │               │
│  │  Chongqing           │               │
│  │  ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓  │ ← 悬停时图片   │
│  │  Advanced synthesis..│    从上滑入    │
│  └──────────────────────┘               │
│  ┌──────┐  ┌──────┐  ┌──────┐          │
│  │生物   │  │调香   │  │植物   │          │
│  │发酵   │  │评香   │  │提取   │          │
│  └──────┘  └──────┘  └──────┘          │
│                                          │
└──────────────────────────────────────────┘
```

**特点**：
- 第一个实验室为**大尺寸 Hero 卡片**（占 2/3 宽度），初始只显示标题和地点
- 悬停时：实验室场景图从上方滑入覆盖背景，描述文字淡入
- 后三个实验室为**小尺寸正方形卡片**，同样悬停揭示
- 所有卡片默认状态干净简约，交互后才展现丰富内容
- **taste-skill 契合**：差异化卡片大小，有惊喜感，去除冗余信息

## 方案 D：「Editorial Feature」编辑精选式

**灵感**：高端杂志排版 + 精致卡片设计

```
┌──────────────────────────────────────────┐
│   R&D Laboratories                       │
│   subtitle                              │
├──────────────────────────────────────────┤
│                                          │
│  ┌──────────────────────────────────┐   │
│  │ ████████████████████████████████ │   │
│  │ ██      大图背景       ██       │   │
│  │ ████████████████████████████████ │   │
│  │  合成香料实验室 · Chongqing      │   │
│  │  Advanced synthesis capabilities │   │
│  │  [Learn More →]                  │   │
│  └──────────────────────────────────┘   │
│                                          │
│  ┌──────┐  ┌──────┐  ┌──────┐         │
│  │ 生物  │  │ 调香  │  │ 植物  │         │
│  │ 发酵  │  │ 评香  │  │ 提取  │         │
│  └──────┘  └──────┘  └──────┘         │
│                                          │
└──────────────────────────────────────────┘
```

**特点**：
- **顶部 Hero**：全宽大幅实验室背景图，标题叠加在图上，带渐变遮罩
- 内容浮在图片底部，包含实验室名称、地点、简短描述和 "Learn More" 链接
- **底部 3 列**：小号竖版卡片，每张带有色顶部装饰条（橙/青/暖橙渐变）
- 图标放在卡片左上角作为视觉锚点
- **悬停**：Hero 卡片图片缩放；小卡片轻微上浮 + 装饰条变色
- **taste-skill 契合**：差异明显的层次结构，顶部 Hero 制造视觉冲击

## 交互与动效建议（所有方案通用）

| 效果 | 说明 |
|------|------|
| **Scroll Reveal** | 模块整体从底部淡入上滑（现有 `.reveal` 复用） |
| **Stagger 入场** | 多卡片依次入场，间隔 100-150ms |
| **悬停过渡** | `0.5s cubic-bezier(0.16, 1, 0.3, 1)` 缓出曲线 |
| **尊重动效偏好** | `@media (prefers-reduced-motion)` 降级为无动效 |

## 选择建议

| 方案 | 适合场景 | 视觉冲击 | 实现复杂度 | 移动端适配 |
|------|---------|---------|-----------|-----------|
| **A - Split Panorama** | 想突出实验室环境氛围 | ★★★★★ | 中 | 右侧 2x2 自然堆叠 |
| **B - Lab Timeline** | 想讲述研发故事线 | ★★★★ | 中高 | 交替变单侧排列 |
| **C - Hover Reveal** | 追求极简 + 交互惊喜 | ★★★★ | 中 | Hero 变全宽堆叠 |
| **D - Editorial Feature** | 传统但精致的升级 | ★★★★ | 低中 | Hero 变自适应高度 |

请选择你感兴趣的方案（可多选），我进一步细化后开始实施。
