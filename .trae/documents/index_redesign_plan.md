# ZOTEQ 主页视觉重构方案

## 📋 项目概述
基于四种设计风格方向，为 `index.html` 打造三套完整的视觉方案。

---

## 🔍 当前状态分析

### 现有设计特征
- **色彩系统**: 橙色 (#e07a5f) 为主，海洋蓝 (#3d8b8b) 为次
- **排版**: Playfair Display (标题) + Inter (正文)
- **组件**: 卡片式设计、毛玻璃效果、圆角设计
- **风格**: 温暖、专业、高品质

---

## 🎨 方案一："Awwwards 现代优雅" 风格

### 1.1 设计理念
获奖级网页设计，追求视觉卓越和高级感。

### 1.2 色彩系统

#### 主色调
| 变量 | 颜色值 | 用途 |
|------|--------|------|
| `--accent-primary` | `#d45d42` | 主要品牌色 |
| `--accent-secondary` | `#2d7a7a` | 次要品牌色 |
| `--accent-tertiary` | `#c9925d` | 装饰点缀色 |

#### 渐变系统
```css
--gradient-hero: linear-gradient(135deg, #2a2a2a 0%, #3d3d3d 50%, #2a2a2a 100%);
--gradient-card: linear-gradient(145deg, rgba(255,255,255,0.95) 0%, rgba(248,248,248,0.9) 100%);
--gradient-glow: radial-gradient(400px circle at 20% 30%, rgba(212, 93, 66, 0.15), transparent 50%);
```

#### 中性色
```css
--bg-elevated: #fefefe;
--text-primary: #1a1a1a;
--text-secondary: #4a4a4a;
--text-muted: #8a8a8a;
--border-subtle: #e6e6e6;
```

### 1.3 排版规范

#### 字体组合
- **标题**: `Playfair Display` (serif) + `Cormorant Garamond` (装饰)
- **正文**: `Inter` (sans-serif)
- **数据**: `Cormorant Garamond` (数字)

#### 字体层级
```css
--text-display: clamp(3.5rem, 8vw, 7rem);
--text-h1: clamp(2.5rem, 6vw, 4.5rem);
--text-h2: clamp(1.75rem, 4vw, 3rem);
--text-h3: clamp(1.25rem, 2.5vw, 1.75rem);
--text-body: clamp(0.95rem, 1.5vw, 1.05rem);
```

#### 排版细节
- **字间距**: 标题 -0.035em，正文 0.01em
- **行高**: 标题 1.1，正文 1.7
- **首字母大写**: 特殊装饰效果

### 1.4 组件设计

#### Hero 区域
- 全屏高度，大字号标题
- 文字叠加在图片上，深色蒙板
- 滚动时文字有视差效果

#### 产品卡片
- 3列网格，鼠标悬停上浮 12px
- 图片裁剪为 4:5 比例
- 信息区域有微妙的垂直线装饰
- 悬停显示 "View Details" 按钮

#### 数据展示
- 大字号数字，Playfair Display
- 单位用小号字体在下方
- 圆形图标背景
- 滚动时计数器动画

#### 时间线
- 垂直居中对齐
- 年份用圆圈显示
- 连线为细的渐变色

### 1.5 交互效果

#### 微交互动画
```css
--ease-elastic: cubic-bezier(0.68, -0.55, 0.265, 1.55);
--ease-smooth: cubic-bezier(0.23, 1, 0.32, 1);
```

#### 悬停效果
- 产品卡片: 上浮 12px + 轻微放大 1.02 倍
- 按钮: 背景色渐变 + 文字上移
- 链接: 文字颜色渐变 + 下划线动画

#### 滚动动画
- 元素从下方淡入
- 有延迟的序列动画
- 数据计数器滚动

### 1.6 响应式布局

#### 断点系统
| 断点 | 宽度 | 网格列数 |
|------|------|---------|
| Mobile | < 640px | 1 |
| Tablet | 640px - 1024px | 2-4 |
| Desktop | 1024px - 1440px | 6-12 |
| Large | > 1440px | 12 |

#### 布局策略
- **Hero**: 全屏高度，自适应
- **产品**: 桌面 3列，平板 2列，移动 1列
- **数据**: 桌面 4列，平板 2列，移动 1列
- **时间线**: 桌面居中，移动向左对齐

## 🎨 方案二："Impeccable 设计系统" 风格

### 2.1 设计理念
基于设计系统的原则，追求一致性和可扩展性。

### 2.2 色彩系统

#### 品牌色彩
```css
--brand-50: #fff1ed;
--brand-100: #ffe0d6;
--brand-200: #ffbaa8;
--brand-300: #ff8c70;
--brand-400: #e07a5f; /* 主色 */
--brand-500: #c4604a;
--brand-600: #9d4a38;
--brand-700: #7a3a2d;
```

#### 海洋色系统
```css
--ocean-50: #eff8f8;
--ocean-100: #d6eeee;
--ocean-200: #a8dcdc;
--ocean-300: #5aadad;
--ocean-400: #3d8b8b; /* 主色 */
--ocean-500: #2d6a6a;
--ocean-600: #1d4a4a;
```

#### 灰色系统
```css
--neutral-50: #faf9f7;
--neutral-100: #f5f3f0;
--neutral-200: #e8e6e2;
--neutral-300: #c8c6c2;
--neutral-400: #a8a6a2;
--neutral-500: #686662;
--neutral-600: #3a3834;
--neutral-700: #1a1a1a;
```

### 2.3 排版规范

#### 字体层级 (8点系统)
| 层级 | 字号 | 行高 | 用例 |
|------|------|------|------|
| Display | 72px | 1.05 | 首页大标题 |
| H1 | 56px | 1.15 | 章节标题 |
| H2 | 40px | 1.2 | 小节标题 |
| H3 | 28px | 1.3 | 卡片标题 |
| Body | 16px | 1.6 | 正文 |
| Small | 14px | 1.5 | 辅助文字 |
| Caption | 12px | 1.4 | 标签说明 |

#### 字体家族
```css
--font-display: 'Playfair Display', 'Cormorant Garamond', serif;
--font-body: 'Inter', system-ui, sans-serif;
--font-mono: 'JetBrains Mono', monospace;
```

### 2.4 组件设计

#### 按钮系统
| 类型 | 样式 | 交互 |
|------|------|------|
| Primary | 品牌色填充，圆角 12px | 上浮 + 阴影增强 |
| Secondary | 边框 + 文字，圆角 12px | 背景变色 |
| Ghost | 透明背景，文字色 | 背景变浅 |
| Text | 仅文字，无背景 | 文字变色 |

#### 卡片系统
| 类型 | 样式 | 用例 |
|------|------|------|
| Product | 白色，阴影，圆角 20px | 产品展示 |
| Feature | 浅灰背景，图标在左 | 特色展示 |
| Timeline | 时间轴连接 | 历程展示 |
| Data | 居中，大数字 | 数据展示 |

#### 导航系统
- 固定顶部，滚动时毛玻璃
- 桌面：水平菜单，4个链接
- 平板：汉堡菜单 + 侧边滑出
- 移动：全屏覆盖菜单

### 2.5 交互效果

#### 动画系统
```css
--duration-fast: 150ms;
--duration-normal: 300ms;
--duration-slow: 500ms;
--duration-very-slow: 800ms;

--ease-out-expo: cubic-bezier(0.16, 1, 0.3, 1);
--ease-out-quart: cubic-bezier(0.25, 1, 0.5, 1);
--ease-in-out: cubic-bezier(0.65, 0, 0.35, 1);
```

#### 交互状态
- Hover: `transform: translateY(-4px);`
- Active: `transform: translateY(-2px);`
- Focus: 清晰的轮廓样式
- Disabled: 透明度降低

### 2.6 响应式布局

#### 网格系统
- 基础: 12列网格
- 间距: 24px (桌面), 16px (移动)
- 容器: 最大宽度 1280px

#### 断点
| 断点 | 宽度 | 布局 |
|------|------|------|
| xs | < 480px | 1列 |
| sm | 480-768px | 1-2列 |
| md | 768-1024px | 2-3列 |
| lg | 1024-1440px | 3-4列 |
| xl | > 1440px | 4列+ |

## 🎨 方案三："Taste 现代简约" 风格

### 3.1 设计理念
基于 taste-skill 原则，避免 AI 俗套设计，追求独特的视觉语言。

### 3.2 色彩系统

#### 非传统色彩
```css
--terracotta: #c4604a;     /* 陶土色 */
--sage: #6a994e;           /* 鼠尾草绿 */
--cream: #faf9f7;          /* 奶油白 */
--charcoal: #2a2a2a;       /* 木炭黑 */
--sand: #d4a574;           /* 沙色 */
```

#### 配色策略
- **主要**: 黑白对比，极少使用彩色
- **点缀**: 小范围使用 terracotta
- **图片**: 高对比度处理

#### 背景纹理
```css
--grain-overlay: url("data:image/svg+xml,%3Csvg...%3E");
--paper-texture: linear-gradient(to bottom, rgba(0,0,0,0.01) 0%, rgba(0,0,0,0.02) 100%);
```

### 3.3 排版规范

#### 字体选择
- **主字体**: `Cormorant Garamond` (高对比度)
- **辅助**: `Inter` (清洁的无衬线)
- **避免**: 过度使用 Inter，寻找更有性格的字体

#### 排版层级
```css
--text-mega: clamp(4rem, 12vw, 10rem); /* 巨大标题 */
--text-large: clamp(2rem, 5vw, 4rem);  /* 大号 */
--text-medium: clamp(1.2rem, 2.5vw, 1.8rem);
--text-small: 0.9rem;
```

#### 排版细节
- **字间距**: 非常宽松或非常紧凑
- **行高**: 1.1 (标题) 或 1.8 (正文)
- **对齐**: 左对齐为主，避免过度居中
- **装饰**: 可选的首字母下沉，下划线装饰

### 3.4 组件设计

#### 不对称布局
- Hero: 文字靠左，图片靠右，不对称
- 产品: 一大两小的不规则网格
- 数据: 2-1-1 布局，打破对称

#### 卡片设计
- 不是所有卡片大小相同
- 没有完全统一的圆角
- 部分卡片有边框，部分没有
- 悬停效果各不相同

#### 图片处理
- 图片有自然的裁剪边缘
- 不是完美的圆角矩形
- 可能有微妙的边框装饰
- 悬停时有轻微的位置移动，不是缩放

### 3.5 交互效果

#### 微妙动画
- 没有夸张的放大效果
- 强调文字的微妙变化
- 滚动时的视差效果
- 颜色的缓慢过渡

#### 避免的俗套
- ❌ 过度使用渐变
- ❌ 相同的卡片悬停效果
- ❌ 紫色/蓝色渐变
- ❌ 完美对称的布局

#### 独特的交互
- 文字位置的微妙调整
- 滚动时的视差背景
- 自定义的鼠标光标
- 页面加载时的文字动画

### 3.6 响应式布局

#### 移动端优先
- 首先设计移动端
- 桌面版是扩展，不是缩小
- 每一个断点都有独特布局

#### 断点
| 断点 | 特点 |
|------|------|
| < 480px | 单列，大图，大文字 |
| 480-768px | 2列，开始不对称 |
| 768-1024px | 3列，复杂布局 |
| > 1024px | 完全不规则网格 |
