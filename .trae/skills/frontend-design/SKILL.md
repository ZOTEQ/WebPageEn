# Frontend Design

Comprehensive frontend design guidelines inspired by https://github.com/anthropics/claude-code/tree/main/plugins/frontend-design/skills/frontend-design.

## Design Fundamentals

### Layout
- Grid systems for consistency
- Flexible and responsive layouts
- Proper spacing and alignment

### Typography
- Clear hierarchy
- Readable fonts
- Appropriate line heights

### Color
- Harmonious palettes
- Accessible contrast ratios
- Semantic color usage

### Components
- Reusable patterns
- Consistent styling
- Interactive states

## Best Practices

### CSS
- Modular stylesheets
- BEM or CSS-in-JS patterns
- Performance optimization

### Accessibility
- WCAG compliance
- Screen reader support
- Keyboard navigation

### Performance
- Lazy loading
- Code splitting
- Optimized assets

## When to Use

- Creating new frontend components
- Reviewing existing code
- Establishing design patterns
- Optimizing frontend performance
- Ensuring accessibility

## Design Checklist

- [ ] Responsive design implemented
- [ ] Color contrast meets AA standards
- [ ] Typography hierarchy is clear
- [ ] Interactive elements have feedback
- [ ] Performance is optimized
- [ ] Code follows best practices
